### Installation

```
pip install pipenv

git clone https://github.com/dhairya137/Django-signin-signup.git login-signup
cd login-signup
pipenv shell
pipenv install
cd loginform
python3 manage.py makemigrations
python3 manage.py migrate
python3 manage.py createsuperuser
python3 manage.py runserver
```
